# Hardware Design

## Schematic Overview
The electronic nose hardware consists of:
- ESP32 microcontroller
- 6x MQ gas sensors (MQ-2, MQ-3, MQ-4, MQ-5, MQ-7, MQ-135)
- Power regulation circuit
- Sensor heating control

## Pin Connections
- MQ-2 → GPIO 34
- MQ-3 → GPIO 35
- MQ-4 → GPIO 32
- MQ-5 → GPIO 33
- MQ-7 → GPIO 25
- MQ-135 → GPIO 26
